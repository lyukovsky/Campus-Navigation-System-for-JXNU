## 📌 项目简介

本项目旨在为 **江西师范大学瑶湖校区** 提供一个简洁、轻量、易于拓展的 **桌面端校园导航系统**。  
使用 **Python + Tkinter** 构建，具备路径规划、节点推荐、地图交互等功能。

---

## ✨ 功能特性

- 🗺 **等比缩放校园地图**，避免图像拉伸  
- 📍 **地图节点加载与路径自动读取**  
- 🔍 **最短路径规划（支持多节点）**  
- ⭐ **智能推荐节点功能**  
- 💬 **全局弹窗居中**  
- 🎨 UI 样式采用 Adobe 风格（深色、现代）
